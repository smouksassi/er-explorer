Interactive exposure-response explorer

Real logistic exposure-response fit (Newton–Raphson IRLS) on an example clinical trial dataset, with a 95% confidence band computed either analytically (Wald) or by seeded case-resampling bootstrap. Select more than one exposure metric to compare panels side by side. Drag a box in a scatter panel to select a subset of patients (selection is linked across panels); click a dose group's box to project the fit across its full observed exposure range (min through max), with its interquartile range emphasized. Save the current view as a reproducible session file and reload it later.

Exposure vs response
Points are jittered vertically for readability and colored by dose group. Dashed grey line is the overall fit; shaded band is the 95% CI. Selecting multiple exposures adds a column per exposure; selecting multiple endpoints adds a row per endpoint, so you can compare how response varies with a given exposure across endpoints - all panels share the same selection. "Fitted probability + CI" marks each active "Group Exposures By" split line with the model's predicted response there in grey, e.g. "83.8 / fit 0.74 [0.70-0.78]" - separate from "Observed % responders", which instead shows the raw (non-model) observed rate per split bin in dark markers, for comparing fitted vs. observed side by side. Clicking a dose row below adds that dose's own observed %/N marker in its own color too ("On dose click" toggle). If markers land close together they automatically stack into a small column with a thin leader line back to each one's true position, so labels never overlap. With exactly one exposure and 2+ endpoints selected, "Compare endpoints" switches to a different view: one panel per endpoint plus a final "(all)" panel overlaying every endpoint's curve together, colored and line-styled by endpoint instead of dose (no raw points in this view, to keep the overlay readable).
Showing all rows
Reference line, group N, and observed-responder split bins are all computed on dosed patients only, excluding placebo (which is fixed at zero exposure by design). Placebo itself still gets its own observed-responder marker at zero exposure when that option is on.
Exposure distribution by dose
Click a row to toggle that dose group's projection onto the fit above. Multiple groups can be active at once. Placebo's exposure is fixed at zero by design, so its row shows only its patient count (N), with no box or distribution shape. The dashed Q1/Q3 lines always mark the same interquartile bounds used for that dose's projection, in both Boxplot and Distribution display modes. "Group N" (Off / N / N (%)) shows how many of each dose's own patients fall in each bin of the "Group Exposures By" split above, once a split is selected.
Session
Every setting above (exposure metric, endpoint, CI method, bootstrap seed, and current selection) is captured in a session file per docs/REPRODUCIBILITY.md — save it, hand it to a colleague, and reloading it reproduces this exact view.

Data: example exposure-response dataset (AUC/CMAX vs ICGI clinical response) bundled for demo purposes. Built directly on @er-explorer/statistical-engine, @er-explorer/visualization-engine, and @er-explorer/session-engine — see docs/ARCHITECTURE.md.
