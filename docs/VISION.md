# Vision

## Mission

Helping scientists answer exposure–response questions through interactive, reproducible and publication-quality visualization and statistical modeling.

Question → Evidence → Decision

Core principles:
- Reproducibility
- Transparency
- Scientific correctness
- Publication-quality
- Interactivity
- Extensibility
- Open source

Long-term vision includes binary, continuous, ordinal and time-to-event models, Clinical Utility Index, dose optimization and reproducible scientific workflows.
